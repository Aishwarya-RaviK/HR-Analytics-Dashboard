# HR Analytics Interactive Dashboard (Excel)

## Project Overview
This repository contains a comprehensive **HR Analytics Dashboard** built entirely in Microsoft Excel. The project demonstrates that complex data narratives and interactive business intelligence can be achieved through advanced spreadsheet modeling, without the immediate need for specialized BI software.

The dashboard provides a 360-degree view of workforce dynamics, specifically focusing on **Employee Attrition** and **Job Satisfaction** to help HR professionals make data-driven retention decisions.

## Key Metrics & KPIs
The dashboard tracks several critical indicators, including:
- **Total Employees:** 1,470
- **Attrition Count:** 237
- **Attrition Rate:** 16.1%
- **Active Employees:** 1,233
- **Average Age:** 37 years

## Dashboard Features & Interactivity
- **Dynamic Filtering:** Includes slicers to filter data by **Department**, **Education Field**, and **Gender**, allowing for granular analysis of specific workforce segments.
- **Attrition Analysis:** Breakdown of attrition by:
    - **Age Band:** (Under 25, 25-34, 35-44, 45-54, Over 55)
    - **Education Field:** (Life Sciences, Medical, Marketing, Technical Degree, etc.)
    - **Job Role:** Identifying high-risk positions like Sales Representatives or Laboratory Technicians.
- **Satisfaction Tracking:** Visualization of **Job Satisfaction** levels across various roles to identify areas for cultural improvement.
- **Marital Status & Gender Insights:** Understanding how demographic factors correlate with retention.

## Technical Implementation
- **Data Modeling:** Processed a dataset of 1,470 records with over 40 attributes.
- **Advanced Formulas:** Used complex logic (including `CF_attrition count` and `CF_age band` calculations) to categorize raw data.
- **Pivot Architecture:** Built the visualization layer using synchronized Pivot Tables and Pivot Charts.
- **UI/UX Design:** Implemented a clean, professional layout with customized color schemes, hidden gridlines, and intuitive navigation.

## How to Use
1. Clone this repository or download the `HR DATA_Excel.xlsx` file.
2. Open the file in **Microsoft Excel 2016 or later** (Desktop version is recommended for optimal performance of slicers and dynamic charts).
3. Interact with the slicers on the dashboard page to explore the data insights.

## About the Project
This project was developed to showcase the power of Excel in the hands of a data professional. It highlights skills in data cleaning, logical modeling, and visual storytelling—fundamentals that are essential for any Data Analyst or MIS Specialist.
